## 🎯 每月完整流程

### 第一步：准备（1 分钟）

**目标**: 获得你当前的真实持仓信息

1. 打开 **Wealthsimple App**
2. 点击 **"My Portfolio"** 或 **"Accounts"**
3. 查看你的 **10 支持仓股票**，记下每支的 **百分比权重**（%）
   
   例如：
   ```
   BMO.TO     15.2%
   MFC.TO     14.8%
   NTR.TO     10.5%
   EMA.TO     12.0%
   ...
   總現金      2.5%
   ```

### 第二步：配置（1 分钟）

**目标**: 告诉 Python 系统你现在持有什么

1. 用 VSCode 打开 `picker.py`
2. **Ctrl+G**（或 **Cmd+G**）跳转到第 **4920** 行
3. 找到这个section：

   ```python
   MY_CURRENT_PORTFOLIO = {
       # "BMO.TO": 0.152,
       # "MFC.TO": 0.148,
       # ... 你的持仓
   }
   ```

4. **删除 `#` 注释符**，填入你的真实持仓：

   ```python
   MY_CURRENT_PORTFOLIO = {
       "BMO.TO": 0.152,      # 15.2%
       "MFC.TO": 0.148,      # 14.8%
       "NTR.TO": 0.105,      # 10.5%
       "EMA.TO": 0.120,      # 12.0%
       "HBM.TO": 0.145,      # 14.5%
       "DIR-UN.TO": 0.070,   # 7.0%
       "ENGH.TO": 0.020,     # 2.0%
       "CNQ.TO": 0.150,      # 15.0%
       "ARX.TO": 0.020,      # 2.0%
       "CAE.TO": 0.020,      # 2.0%
   }
   ```

   ✅ **权重填写规则**：
   - Wealthsimple 显示的百分比 ÷ 100 = 字典中的权重
   - 示例：15.2% → `0.152`
   - ⚠️  所有权重总和应 **接近 1.0**（允许 0.95-1.05）
   - 如果持仓少于 10 只？直接写你实际持有的即可

5. **保存文件** （Ctrl+S 或 Cmd+S）

### 第三步：运行（5 分钟）

**目标**: 让系统自动分析并生成建议

1. **打开终端**（VSCode 里 Ctrl+`)
2. **切换到项目目录**：

3. **运行系统**：

   - ✅ 生成 PDF 报告并邮件发送

   **进度指示**：
   ```

   [2/4] 约束过滤 ...           ⏱️  30 秒
   [3/4] 特征工程 + 模型 ...     ⏱️  2-3 分钟
   [4/4] Walk-Forward 回测 ...  ⏱️  0.5-1 分钟
   ━━━━━━━━━━━━━━━━━━━━
   ```

4. **去冲杯咖啡**☕ 等着（不需要看屏幕）

### 第四步：执行（1-2 分钟）

**目标**: 根据报告建议调仓

1. **打开你的邮箱**：
   - 地址：`carlchenn@hotmail.com`
   - 寻找标题：`[TSX选股] 当月建议配置 - 2026-04-26`

2. **下载并打开 PDF 报告**

3. **找到这个表格**（"Position Sizing" 或 "仓位建议"）：

   ```
   │Stock │    │  Target │ Current  │Shares to  │
   ```

## 🔧 配置项详解

### MY_CURRENT_PORTFOLIO（核心配置）

**位置**: `picker.py` 第 4920 行

**用途**：
- 告诉系统你当前持有什么股票，占比多少
- 用于计算"缓冲带"（避免频繁换仓）
- 用于 Black-Litterman 优化（考虑现有仓位）

**更新频率**：
- ✅ **必须**：每月运行前更新
- ❌ **不要**：每周更新（会导致数据不一致）

**常见错误**：

| 错误 | 后果 | 解决 |
|------|------|------|
| 权重不更新 | 选股加权与实际仓位脱离 | 影响 BL 优化结果偏离 |
| 权重总和 > 1.1 | Python 会警告 | 检查持仓合计，重新计算比例 |
| 持仓股票遗漏 | 缓冲带失效 | 选股可能提出过度调整 |

### MODE（运行模式）

**位置**: `picker.py` 第 4904 行

**三种模式**：

1. **"pick"** （推荐 ⭐）
   - 用途：仅当月选股 + BL 优化
   - 耗时：5 分钟
   - 输出：JSON + PDF 报告
   - **默认值，99% 情况下用这个**

2. **"backtest"**
   - 用途：历史回测（过去 12 个月逐月模拟）
   - 耗时：25-30 分钟
   - 输出：完整的月度 P&L 曲线、Sharpe Ratio、Max Drawdown
   - **用来验证系统有效性** （建议每季度跑一次）

3. **"both"**
   - 用途：先回测再选股
   - 耗时：30-35 分钟
   - 输出：完整回测报告 + 当月选股建议
   - **推荐周末或假期前跑** （耗时较长）

**修改方式**：
```python
# 只改这一行
MODE = "pick"    # 改成 "backtest" 或 "both"
```

### EMAIL_CONFIG（邮件配置）

**位置**: `picker.py` 第 4940 行

**作用**：自动生成 PDF 报告并发送到邮箱

**当前配置**：

| Gmail 显示"登录失败" | 确认 app password 正确（不是真实密码） |

**唯一允许的修改**：
- ✅ MY_CURRENT_PORTFOLIO（持仓更新）

- ✅ EMAIL_CONFIG（邮箱配置）

## 🚨 故障排除

### 问题 1：运行时显示 "KeyError: XIU.TO"

**原因**：数据下载中断

**解决**：
```bash
# 清空缓存
rm -rf ./simfin_data
# 重新运行
python picker.py
```

### 问题 2：报告没有发送到邮箱

**排查清单**：
1. 确认 `EMAIL_CONFIG` 的 `to` 邮箱地址正确
2. 确认 Gmail app password 是否过期（3 个月）
3. 检查网络连接（VPN/防火墙）
4. 查看终端输出，看是否有错误信息

### 问题 3：权重和 > 1.0 或 < 0.95

**原因**：
- Wealthsimple 显示的百分比有舍入
- 你的持仓不恰好 10 只股票
- 账户有现金未投

**解决**：
- 权重和 0.98-1.02 是正常的，系统会自动处理
- 如果 > 1.05 或 < 0.90，重新检查百分比计算

### 问题 4："模型预测全 NaN"

**原因**：特征计算出错（通常是财报数据缺失）

**解决**：
```bash
# 强制更新财报缓存
rm -rf ./simfin_data/fmp_cache.pkl
python picker.py
```

   - 每月 25-28 日之间更新一次

### ❌ 不要这些事

1. **频繁运行系统**
   - 不要一天运行多次（接近数据会改变预测）
   - 最多每周一次（基于周末收盘数据）

2. **修改 CONSTRAINTS 或特征**
   - 除非你理解底层数学
   - 一个小改可能导致整套系统失效

3. **忽视缓冲带**
   - 系统设计了 ±12 档的"缓冲带"防止过度换仓
   - 不要看到不同就立刻调整（会产生手续费和滑点）

4. **依赖"最近 1 个月"的收益**
   - 量化选股是 6-12 个月的策略
   - 短期波动正常，不要惊慌调整
